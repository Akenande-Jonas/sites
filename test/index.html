TP1.1 Front avancé

Introduction

Dans cet exercice, vous allez créer une page HTML simple avec un formulaire de contact. Le formulaire comprendra des champs pour le nom, le prénom, l'email et un message. Vous ajouterez également un bouton pour soumettre le formulaire.

Exercice 1

Créer une page HTML avec un formulaire de contact comprenant les champs suivants :

  • Nom
  • Prénom
  • Email
  • Message

Ajouter un bouton "Envoyer" pour soumettre le formulaire.













Note: Le formulaire n'a pas de fonctionnalité de soumission réelle, car l'action est définie sur "#".
